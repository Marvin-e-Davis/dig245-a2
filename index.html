How to find the source code on my favorite website! (on windows chrome)

AKA How to break the developer's defenses.

  1. Go to my favorite website: https://www.wcostream.com/
  2. Hit the three vertical dots on the top right of the screen
  3. Hover over the option "More tools"
  4. Click developer tools to view the HTML source code

Shortcut: Hit F12 on your computer
